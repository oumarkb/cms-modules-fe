Module Description

This module is designed to display a clear and modern directory of registered members on your website. It's perfect for communities, associations, or any site that requires a professional presentation of its members.

Key Features:

  • Real-time Search: Quickly find members by searching names or email addresses using an intuitive search bar.
  • Clean and Responsive Layout: A modern design optimized for all devices, ensuring the best user experience.
  • Comprehensive Member Details: Each member is displayed with their name, email, phone number, address, and region.
  • Integrated Statistics: Display the total number of users for an instant overview.

Frequently Asked Questions

To update a member's information, follow these steps:

  1. Log in to your Joomla admin panel
  2. Set up the User Profile fields:
    • Navigate to System > Plugins
    • Search for "User - Profile" (this is a built-in Joomla plugin)
    • Enable the plugin by clicking its status icon
    • Click on the plugin name to configure the profile fields:
      • Set each field as Optional, Required, or Disabled based on your needs
      • Save your configuration
    • These fields will now appear in the user registration form and profile editing page
  3. To update a specific member's information:
    • Go to Users > Manage
    • Find and click on the user you want to update
    • You'll find their information in two sections:
      • Account Details: Basic information (username, name, email)
      • User Profile: Additional information (address, phone, etc.)
    • Make your changes and click "Save"
