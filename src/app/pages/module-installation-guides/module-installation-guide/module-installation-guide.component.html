Steps to Install an Xcelerate Joomla Module

  1. If you haven't bought a license yet for the module, make sure to do it on main page. You'll need the license key to activate the module.
  2. Download the module from this repository. (Make sure to take the last version)
  3. Go to your Joomla admin panel, navigate to System > Install > Extensions.
  4. Drag and drop or browse to the zip module you downloaded to install it.
  5. Navigate to Content > Site Modules, search the module and click on it.
    • On Module tab, make sure to have the Position and Status set to your preference.
    • On Menu Assignment tab, make sure to select the menu items where you want the module to appear.
    • On Module Configuration tab, fill in the required fields, save and close. The same license key can't be used on multiple domains. Make sure to use the correct one.

You should now see the module displayed on the concerning pages.
