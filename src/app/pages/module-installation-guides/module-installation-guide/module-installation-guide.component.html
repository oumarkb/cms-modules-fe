1. If you haven't bought a license yet for the {{moduleName}} module, make sure to do it on main page. You'll need the license key to activate the module.
  2. If you haven't downloaded the {{moduleName}} module from Joomla Extensions Directory yet, do it on this repository. (Make sure to take the last version)
    • Once the module downloaded, go to your Joomla admin panel, navigate to System > Install > Extensions.
    • Drag and drop or browse to the zip module you downloaded to install it.
  3. Navigate to Content > Site Modules, search the module and click on it.
    • On Module tab, make sure to have the Position and Status set to your preference.
    • On Menu Assignment tab, make sure to select the menu items where you want the module to appear.
    • On Subscription tab, fill in the required fields.
    • Click on Save & Close button to save your changes.
