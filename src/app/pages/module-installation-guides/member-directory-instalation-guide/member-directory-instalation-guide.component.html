1. Make sure to enable the native Joomla User - Profile plugin. This is required for the module to work properly.
    • Navigate to System > Plugins
    • Search for the User - Profile plugin and click on it.
    • Make sure to have the Status set to Enabled and activate the fields you want to use.
    • Click on Save & Close button to save your changes.
  2. Navigate to Content > Site Modules, search the module and click on it.
  3. On Fields Configuration tab, you can add, edit, remove, or rearrange the fields as you like. Only the fields defined in Joomla's native 'User - Profile' plugin are supported.

    This module supports only fields declared in the native Joomla User - Profile plugin.
    If you want to add others custom fields, you can do it by storing the values into #__user_profiles table of your database.

    For more icons, please visit Font awesome

  4. Click on Save & Close button to save your changes.

You should now see the module displayed on the concerning pages.
